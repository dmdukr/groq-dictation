## Session-based Normalizer (implemented earlier, verified working)
- Persistent LLM conversation — промпт відправляється один раз
- Context зберігається між диктовками
- Handoff при 80% контексту (summary + golden texts)
- Один прохід замість двох = вдвічі швидше

## Architecture Decisions

### Translation Engine
- DeepL primary (кращий за якістю, 2x менше правок ніж Google)
- 5 ключів з ротацією = 2.5M chars/month free
- Groq LLM fallback коли DeepL недоступний

### Update Mechanism
- Onedir PyInstaller (без temp extraction)
- Updater качає setup installer з GitHub releases
- /VERYSILENT /SUPPRESSMSGBOXES для оновлень

### Build Pipeline (feedback saved to memory)
1. /simplify (3 parallel review agents)
2. ruff check + bandit security
3. Build on local drive (--distpath C:/tmp/groq-dist)
4. Copy dist, build installer with ISCC
5. Commit, tag, push, gh release create

### Browser Extension (planned, not implemented)
- Chrome/Edge/Brave/Vivaldi/Firefox extension
- WebSocket з'єднання з додатком (localhost:19384)
- Content script: DOM text extraction/replacement
- Toggle original/translated
- Installed via Settings → "Install Browser Extension"
